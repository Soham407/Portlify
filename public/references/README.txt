PORTFOLIO TEMPLATE REFERENCE DESIGNS
=====================================

These are the inspiration sources for the 5 portfolio templates implemented in this project.

1. GLASS TEMPLATE (id: "minimal")
   Inspired by: Modern glassmorphism / dark SaaS landing pages
   Color: Dark purple-indigo gradient (#0f0c29 → #302b63)
   Accent: Violet #a78bfa + Blue #60a5fa
   Features: Frosted glass cards, avatar with glowing ring, blur-fade animations, pill skill tags

2. NIGHT OWL TEMPLATE (id: "developer")
   Inspired by: Angel Garcia portfolio + Arthur Araujo portfolio
   Color: GitHub dark (#0d1117), Card: #161b22, Border: #30363d
   Accent: Teal #2DD4BF
   Features: Sticky monospace navbar, split hero (text left/avatar right), project rows with gradient thumbnail,
             vertical timeline experience, tech tag badges styled like code

3. VIBRANT TEMPLATE (id: "creative")
   Inspired by: Moskur portfolio (moshurfolio.com-style)
   Color: White bg, Black dark sections
   Accent: Lime-yellow #CBFF4D
   Features: Bold uppercase hero, lime pill headline badge, scrolling marquee strip, emoji skill cards,
             colorful project cards (purple/yellow/green/red/blue), bold CTA footer

4. EDITORIAL TEMPLATE (id: "corporate")
   Inspired by: Elian Ross portfolio (elianross.com-style)
   Color: Off-white/cream #FAFAF7
   Accent: Green #16A34A
   Features: Availability badge, huge display typography (8xl), split about section,
             bento project grid (first card large + full width), numbered experience list, "Let's Make It Happen" CTA

5. BRUTALIST TEMPLATE (id: "photography")
   Inspired by: Devanta Ebison portfolio design exploration
   Color: Pure black #000000, pure white text
   Accent: None — only white, gray shades
   Features: Massive condensed hero type (clamp 3rem–10rem), grayscale avatar, dot-separated skill inline list,
             project rows with numbered index, clip-path text reveal animations, raw "GET IN TOUCH →" CTA

DESIGN PRINCIPLES USED:
- No resume-style card grids — each section has its own unique layout treatment
- Strong visual identity through color palette + typography choices
- Template-specific animations (not just CSS class swaps)
- Mobile-responsive with thoughtful breakpoints
- All templates share the same PortfolioData interface for data compatibility
